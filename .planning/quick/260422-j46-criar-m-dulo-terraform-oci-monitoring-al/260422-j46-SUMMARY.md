# Phase quick Plan 260422-j46: OCI Billing Alarm Module Summary

**One-liner:** OCI Monitoring alarm on `oci_billing` namespace that fires CRITICAL alert via ONS email when `CostByService[1d].sum() > 0`, protecting the Always Free tier.

## What Was Built

New Terraform module `oci-billing-alarm` under `terraform/infra/modules/` that provisions:

1. **ONS Notification Topic** (`assessforge-billing-alerts`) — separate from Cloud Guard alerts topic
2. **Email Subscription** (conditional on `notification_email != ""`) — routes alerts to operator email
3. **OCI Monitoring Alarm** (`assessforge-billing-cost-alarm`) — queries `oci_billing` namespace at tenancy level, fires CRITICAL when any cost appears, 1-hour pending duration, daily repeat

The module is integrated into the root infra layer with a count-based enable guard (`enable_billing_alarm = false` by default).
