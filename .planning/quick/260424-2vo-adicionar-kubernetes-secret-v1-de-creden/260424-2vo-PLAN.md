<objective>
Wire the GitHub PAT for the private `gitops-setup` repo into ArgoCD as a Kubernetes Secret so ArgoCD can sync the `bootstrap` Application — closing the classic GitOps chicken-and-egg gap (ESO/ExternalSecret that would otherwise manage this PAT lives *inside* the repo ArgoCD cannot yet read).

Purpose: Unblock ArgoCD sync which is currently failing with `authentication required: Repository not found`. The Terraform-created Secret serves as the bootstrap seed; once the gitops-setup repo syncs, ESO + ExternalSecret take over PAT rotation via OCI Vault.

Output: One new `kubernetes_secret_v1.gitops_repo_creds` resource in the `oci-argocd-bootstrap` module, a new module variable `gitops_repo_pat`, root-level wiring, and a short operator note in the Cloud Shell runbook.
</objective>
