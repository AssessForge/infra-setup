<objective>
Convert the single-string `notification_email` configuration into a `list(string)` named `notification_emails` so operators can notify multiple recipients on both the Cloud Guard ONS topic and the billing alarm ONS topic via `for_each` fan-out.

Purpose: Only one recipient can currently receive OCI notifications. Operators want all team members to receive Cloud Guard and billing alerts without managing a distribution list outside OCI.

Output: Renamed variable (plural, `list(string)`), two refactored modules using `for_each = toset(var.notification_emails)` on their `oci_ons_subscription` resources, updated root wiring, and an updated `terraform.tfvars.example` showing the new list syntax.
</objective>
